Welcome to the Animated Menu

Click the menu button in the top-left corner to open the animated navigation menu. Toggle between light and dark modes using the theme button in the top-right corner.

The background features an orthogonal grid pattern that adapts to the current theme. You can also drag the menu left to close it!

Features

  • • Cinematic spring-based animations
  • • Drag to close functionality
  • • Staggered menu item animations
  • • Close button with rotating animation
  • • Responsive design for all screen sizes
  • • Light and dark mode support
  • • Orthogonal grid background
  • • Interactive hover effects
